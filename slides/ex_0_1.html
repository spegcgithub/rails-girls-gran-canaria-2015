Bentobox - Solución

  1. PHP

    Lógica. Lenguaje de programación.

  2. SaaS

    Modelo de negocio. Software as a Service.

  3. RoR

    Lógica. Ruby on Rails. Framework.

  4. HTML

    Estilo y estructura. Lenguaje de marcado.

  5. MongoDB

    Base de datos. NoSQL.

  6. Apache

    Infraestructura. Software de servidor de código abierto.

  7. CSS

    Estilo y estructura. Define la apariencia de la web.

  8. Django

    Lógica. Framework. Equivalente a Rails para Python.

  9. MySQL

    Base de datos. Sistema de base de datos de código abierto.

  10. Nginx

    Infraestructura. Software de servidor de código abierto.

Front-End y Back-End

  • Front-End

    Parte de la web que podemos ver y con la que se puede interactuar.

    Diseño, HTML, CSS, JavaScript, fuentes, botones, formularios...

  • Lógica - Backend-End

    Parte encargada de procesar la información y almacenarla.

    Ruby, RoR, Python, Django, PHP, ...

  • Base de datos - Backend-End

    "Almacén" de información

    PostgreSQl, MySQL, SQLite, ...

  • Infraestructura - Backend-End

    Servidor web.

    Apache, Nginx, IIS, ...

¿Qué necesitamos para empezar?

Ordenador

Con el entorno ya instalado. En este caso, la máquina virtual.

Editor de Texto

Sublime Text, Vim, Emacs y Gedit son ejemplos de editores de texto que se pueden usar para escribir código y editar archivos.

Terminal

Es el lugar donde se inicia el servidor de Rails y se ejecutan los comandos.

Navegador web

(Firefox, Safari, Chrome) para vizualizar la aplicación.

Aplicación «Ideas»

Aplicación donde podamos anotar nuestras ideas y asociarles imágenes.
